# Overview

The is a simple python socket program to demonstrate the skills a learnt in the networking module of the CSE310 course.
This is a a chatroom of a client-server model: the server listen for connection and many clients can connect and send messages which the server then distributes to all connected clients. NOTE: this can only work when the server and clients are on the same network.
